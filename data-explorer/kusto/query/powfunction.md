# <a name="pow"></a>pow()

累乗の結果を返します

## <a name="syntax"></a>構文

`pow(`*ベース* `,`*指数*`)`

## <a name="arguments"></a>引数

* *base*: base 値。
* *指数*: 指数部の値。

## <a name="returns"></a>戻り値

べき乗指数 (base ^ 指数) に底を累乗した値を返します。
