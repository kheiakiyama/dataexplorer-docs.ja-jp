# <a name="gamma"></a>gamma()

[ガンマ関数](https://en.wikipedia.org/wiki/Gamma_function)を計算します

## <a name="syntax"></a>構文

`gamma(`*閉じる*`)`

## <a name="arguments"></a>引数

* *x*: ガンマ関数のパラメーター

## <a name="returns"></a>戻り値

* X のガンマ関数。
* ログガンマ関数の計算については、「 [loggamma ()](loggammafunction.md)」を参照してください。
