# <a name="exp2"></a>exp2()

X の底2の指数関数。2は、power x: 2 ^ x に発生します。  

## <a name="syntax"></a>構文

`exp2(`*閉じる*`)`

## <a name="arguments"></a>引数

* *x*: 指数部の実数部の値。

## <a name="returns"></a>戻り値

* X の指数値。
* 自然対数 (底 2) の場合は、「 [log2 ()](log2-function.md)」を参照してください。
* 指数関数 (底 e と底10の対数) については、「 [exp ()](exp-function.md), [exp10 ()](exp10-function.md) 」を参照してください。
