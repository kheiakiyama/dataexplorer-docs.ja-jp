# <a name="kusto-explorer-code-navigation"></a>Kusto エクスプローラーのコードナビゲーション

Kusto. エクスプローラーには、クエリシンボル情報を使用した簡単なコードナビゲーションのための機能がいくつか用意されています。

## <a name="go-to-symbol-definition"></a>シンボル定義への移行

現在のシンボルの定義に移動するには、 `F12` またはショートカットを使用し `Alt` + `Home` ます。

## <a name="list-all-references-of-a-symbol"></a>シンボルのすべての参照を一覧表示する

ショートカットを使用して、現在のシンボルのすべての参照を取得でき `Ctrl` + `F12` ます。
