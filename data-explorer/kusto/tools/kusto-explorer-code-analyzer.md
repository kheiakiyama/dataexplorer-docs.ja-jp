# <a name="kusto-explorer-code-analyzer"></a>Kusto Explorer コードアナライザー

Kusto. エクスプローラーには、現在のクエリを分析し、適用可能な改善に関する推奨事項のセットを出力するコードアナライザーユーティリティが用意されています。 

を使用して `Ctrl` + `F6` 、アクティブなクエリでアナライザーを実行します。
